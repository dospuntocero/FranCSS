What is FranCSS?

FranCSS (pronounced as Francis) it's a new wireframing html/CSS Framework. It's really tidy and simple but can make the difference for ultra fast website mockups

You can also use it as foundation for your real website

How it works

FranCSS uses 4 different base less files for creating the complete experience:

reset.less
based on yui reset
mixins.less
that contains all the trics for supporting css3, vendors, etc
layout.less
which has all the layout information
typography.less
which has all the content style definitions
